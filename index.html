🔧 One-Time Setup
  1. Go to console.firebase.google.com → Create a free project
  2. Enable Firestore Database → Start in test mode
  3. Project Settings ⚙️ → Your apps → Web app → Copy config
  4. Open this HTML file → find const firebaseConfig = {…} → paste your values
  5. Re-deploy to Vercel → Admin → 📊 Database → 🚀 Force-Save ALL
